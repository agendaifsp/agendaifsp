Agenda Adaptada:
Uma agenda web interativa e responsiva com elementos de gamificação, projetada para oferecer uma experiência de organização pessoal leve, intuitiva e motivadora.

Sobre o Projeto
O Minha Agenda vai além de um simples calendário. Ele combina a funcionalidade de visualização de datas com um painel de progresso inspirado em sistemas de RPG (XP, Níveis e Missões), incentivando o usuário a manter sua rotina em dia de forma divertida.

Principais Funcionalidades:
Calendário Dinâmico: Navegação entre meses e destaque automático para o dia atual.

Sistema de XP e Nível: Barra de progresso visual para acompanhar a evolução do usuário.

Painel de Missões: Monitoramento de tarefas diárias com indicadores visuais de conclusão.

Design Responsivo: Interface adaptável para dispositivos móveis e desktops.

Sidebar Intuitiva: Navegação rápida para diferentes seções do app.

Tecnologias Utilizadas
O projeto foi construído utilizando o conceito de Clean Code e tecnologias web nativas para garantir performance:

HTML5: Estrutura semântica.

CSS3: Estilização moderna com Flexbox, Grid e Variáveis, além de fontes do Google Fonts (Nunito).

JavaScript: Lógica do calendário e manipulação dinâmica do DOM sem dependências externas.
